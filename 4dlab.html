🔥 Configurar Firebase
Para sincronizar datos entre todos los dispositivos
📋 Pasos para obtener las credenciales:

1. Ve a console.firebase.google.com
2. Crea un proyecto nuevo (gratis) o usa uno existente
3. Ve a Realtime Database → Crear base de datos → Modo de prueba
4. Regresa al inicio del proyecto → Agregar app web (</>)
5. Copia los valores del objeto firebaseConfig y pégalos abajo
💡 ¿Ya configuraste Firebase antes? Los datos de configuración se guardan en este navegador. Si abres el archivo en otro dispositivo, tendrás que ingresar las credenciales una sola vez en cada dispositivo.
